Hjälp & Information (Rapport)

Installera Appen

Liknande instruktioner som för Kortleks-appen för att lägga till på hemskärmen.

Grundläggande Användning

1. Lägg till Skott: Ange klockslag (standard är nuvarande tid), markera om skottet är verifierat, och klicka "Lägg till skott".

2. Hantera Rapportlista: Varje skott visas i listan.

  • Lägg till Fällt Vilt: Klicka på plus-ikonen (+) för ett skott för att öppna ett formulär där du kan ange viltslag, pass/plats, skytt, och köttfördelning.
  • Redigera Fällt Vilt: Om vilt redan är tillagt, klicka på penn-ikonen för att redigera detaljerna.
  • Ta bort Rapport: Klicka på soptunna-ikonen för att ta bort hela rapportposten (skott och eventuellt vilt).

3. Datahantering:

  • Spara Data: All data sparas automatiskt i din webbläsare. Den finns kvar även om du stänger och öppnar appen igen på samma enhet.
  • Exportera Data: Klicka på "Exportera Data (CSV)" för att ladda ner all rapportdata som en CSV-fil. Denna fil kan öppnas i kalkylprogram.
  • Rensa Lista: Klicka på "Rensa Hela Listan" för att ta bort all sparad data. Detta går inte att ångra.

4. Röstinmatning: Använd mikrofonikonerna för att tala in information för "Skytt" och "Köttfördelning" när du lägger till/redigerar fällt vilt.
